Delete confirm

Delete confirm is an trash icon component.

Contents

  • Usage
  • Examples

Usage

            import { NgxDeleteConfirmModule } from 'ngx-delete-confirm';

            @NgModule({
                imports: [NgxDeleteConfirmModule.forRoot(),...]
            })
            export class AppModule(){}

Examples

Default use

Default usage.

  • component 


        
  1. <ngx-delete-confirm (onConfirm)="sendDelete($event)" (onCancel)="sendCancel($event)"></ngx-click-to-edit>

  1. import { Component } from '@angular/core';
  2.  
  3. @Component({
  4.   selector: 'demo-delete-confirm-default',
  5.   templateUrl: './default.html'
  6. })
  7. export class DemoDeleteConfirmDefaultComponent {

  11.     this.msg = 'Delete it.'

  15.     this.msg = 'Backoff!'
  16.   }
  17. }
